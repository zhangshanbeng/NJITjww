# 功能说明
爬取南京工程学院教务网通知（不需要内网权限的通知），通过设置触发器，实现定时发送爬取结果
# 所需Python包  
* bs4
* requests
* smtplib
* email  
# 运行环境  
* Python 3.6  
# 执行方法  
NJITjww.main  
# 邮件发送说明
`password`为你邮箱的smtp授权码  
# 个人触发器设置思路  
* 12：00 教务网一般在12点左右发布通知，保证及时性
* 14：30 过了这个点学校一般不发通知了
* 23：55 怕漏，保证通知不漏
